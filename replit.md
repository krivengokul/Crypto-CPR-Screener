# CPR Crypto Screener

Crypto CPR screener built with React and Vite.

## Run & Operate

- `pnpm --filter @workspace/cpr-screener run dev` - run the CPR screener app
- `pnpm run typecheck` - typecheck the app
- `pnpm run build` - build the app

## Stack

- pnpm workspaces, Node.js 24, TypeScript 5.9
- React + Vite
- GitHub Pages deployment from `artifacts/cpr-screener`
